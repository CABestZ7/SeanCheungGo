Computational Thinking 计算思维

计算机的基本结构首先是需要了解的，在当下的时代我们有许多种类的计算机，比如说：台式电脑（Desktop）、笔记本（Laptop）、平板电脑（Tablet）等等，他们的大小和基本功能不太一样，但他们的基本架构是差不多相同的。

计算机结构

他们都有CPU（Central Processing Unit）、RAM（Random Access Memory）和存储，即Storage（包括硬盘和内存卡），而这些就被称为计算机结构。

那么这个结构到底是怎么样的呢？就不得不提到计算机结构模型，插一句，现代计算机其实又被称为冯·诺依曼结构计算机，是的，当代计算机就是以冯·诺依曼结构为基础设计的， 回归原题，计算机结构实际上遍被称为冯·诺依曼结构。

冯·诺依曼结构（The Von Neumann Model）

  • 中间那个巨大的长方形是计算机本身：
    • 每个计算机都有一个CPU，其中包含又一个控制单元（Control Unit）和一个算数逻辑单元（ALU，即，Arithmetic Logic Unit)，而最下面有一个内存单元（Memory Unit），它是用来存储数据和指令的。
      • 控制单元（Control Unit）：管理程序和应用如何运行的，它分配系统资源，并在系统组件之间进行协调。
      • 算术逻辑单元（ALU）：一个处理数据的“大型计算器”。
        形象一点，如果将CPU比作一个大家族，那么控制单元就是一个这个家族的大管家，算术逻辑单元勉强可以算作这个家族的幕僚，是诸葛亮型人物。
      • 内存单元（Memory Unit） 会与CPU进行数据和指令的交换，也就是他们之间的箭头。通常而言，内存单元指的是RAM（随机存取存储器），暂时存储一些临时程序和数据。
  • 我们通过输入设备（Input）将数据和指令输入计算机，比如摄像机、扫描仪、键盘、写字板等。
  • 在处理数据时，计算机可以在输出设备（Output）上显示结果或执行一些其他任务，而输出设备是用来接收计算机数据的输出的，比如显示器、打印机、绘图仪、扬声器等等。

What & Why Java

Java是什么

Java是一门面向对象的编程语言，开发者为James Gosling（詹姆斯·高斯林）。

  • Java特点：面向对象编程（OOP）：Object-Oriented Programming

Java作为一门超受欢迎的编程语言，本身是由Sun微系统公司研发的，刚开始Java还不叫Java叫做Oak，只不过后来Oracle收购了Sun。目前Java由Oracle对Java语言进行更新维护。 Java之所以受欢迎，有一点非常吸引人：

  • Java特点：平台独立性（Platform Independence）
    • 这意味着，Java语言可以在许多地方，从电脑端到移动端皆可运行，而不需要像其他编程语言一样还需要对代码进行修改。
      Java生态：火星车（Mars rover）、手机（Cell phones）、蓝光光盘（blu-ray disc）、网站服务端（Web server）、超级计算机（Supercomputing）
    • 这也就产生了一个说法，写就写一次，我到处运行（Write Once, Run Everywhere）。

Java的优势所在（Advantages）

  1. Higher cross- functionality and portability （更高的跨功能和可移植性）
    写就写一次，我到处运行。
  2. OOP, distributed, supports multithreading, offers multimedia and network support （面向对象的、分布式的、支持多线程的、支持多媒体和网络的）
  3. mature, therefore more stable and predictable（一门成熟的语言，因此更加稳定和可预测）
  4. large active user community and support available（拥有大量活跃的用户社区和可用的支持）
  5. Java programs are compiled independent of platform in bytecode language（Java程序是用字节码语言独立于平台进行编译的-平台独立性）
    这种语言允许相同的程序在任何安装了JVM的机器上运行(run on any machine that has a JVM installed)
  6. tons of frameworks（java有大量框架）
