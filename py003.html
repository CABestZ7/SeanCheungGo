创建变量 Creating Variables

变量时一种用来储存数据值的容器。 跟其他编程语言不同，python没有用来声明变量的命令，只有首次为其赋值时，才会创建变量。
同时，python变量不需要使用任何特定类型声明，甚至于可以在变量被创建之后更改其类型。

python是一种动态语言，即一种变量本身类型不固定，相应的是静态语言，即在定义变量时需同时指定变量类型
= 与 == 是不同的，前者是赋值语句，后者是等值判断。

变量名称 Variable Names

  • 命名规则
    • 变量名必须以字母或下划线字符开头
    • 变量名称不能以数字开头
    • 变量名只能包含字母数字字符和下划线（A-z、0-9 和_）
    • Case-Sensitive：区分大小写
  • 一些命名形式
    1. Camel Case： aVariableName = 'tim'
    2. Pascal Case： MyVariableName = 'tim'
    3. Snake Case： my_variable_name = 'tim'

  2. 在内存中创建一个名为 x 的变量，并将它指向 apple 而当将变量赋值给变量时：

为什么此处的输出是A呢？

  1. 执行line1时，解释器创建字符串 'A' 和变量 a ，并将 a 指向 A
  2. 执行line2时，解释器创建变量 b ，并将 b 指向 a 指向的字符串 'A'
  3. 执行line3时，解释器创建字符串 'APPLE' ，并将 a 指向新的字符串 APPLE ，但 b 并没有更改

全局变量与局部变量 Global Variables & Local Variables

  • 全局变量 Global Variables：

    在函数外部创建的变量，内外部都可以使用。
    global关键字：在函数内部创建或更改全局变量。

  • 局部变量 Local Variables：

    在函数内部创建的变量，只能在内部使用，使用时不影响该函数外部的变量。
    倘如在内部创建一个与全局变量同名的变量，全局变量在该函数内的值将被覆盖。

Casting

当需要为变量指定类型时，可以通过casting来完成，casting时使用构造函数来完成转换：

  • int() 用整数字面量、浮点字面量构造整数（通过对数进行下舍入），或者用表示完整数字的字符串字面量
  • float() 用整数字面量、浮点字面量，或字符串字面量构造浮点数（提供表示浮点数或整数的字符串）

  • str() 用各种数据类型构造字符串，包括字符串，整数字面量和浮点字面量

常量 Constants

常量实际上也是一种变量，创建一个变量，将变量指向一个固定的不再需要更改的数据值，即常量。
常量常命名为全大写，如 PI = 3.1415926

事实上，python中也没有一个数据类型为常量的，因此并没有任何手段去限制这个常量不被改变。
