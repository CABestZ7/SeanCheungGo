计算机可以处理数据，变量也可以储存数据，数据有不同类型，不同类型的数据可以执行不同的操作。
数据类型可以看作是一个类，每一种数据类型都是一个对象，都具有自己的属性与方法。

数值 Number

数值主要包含三种类型：

  1. int 整数
    • 整数是完整的数字，正数或负数，没有小数，长度不限。
    • python可以处理十六进制，有时使用十六进制表示整数比较方便，十六进制用0x前缀和0-9，a-f表示，例如：0xff00，0xa5b4c3d2，等等。
    • 对于很大的数，例如10000000000，很难数清楚0的个数。Python允许在数字中间以_分隔，因此，写成10_000_000_000和10000000000是完全一样的。十六进制数也可以写成0xa1b2_c3d4。
  2. float 浮点数
    • 浮动或"浮点数"是包含小数的正数或负数。
    • 浮点数也可以是带有"e"的科学数字，表示 10 的幂，把10用e替代，1.23x109就是1.23e9，或者12.3e8，0.000012可以写成1.2e-5，等等。
  3. complex 复数 复数用 "j" 作为虚部编写，如 x=3+5j。
整数和浮点数在计算机内部存储的方式是不同的，整数运算永远是精确的，而浮点数运算则可能会有四舍五入的误差。

类型转换 Type Conversion

可以通过构造函数实现三种类型之间的转换，但要注意的是，复数无法被转换为其他数字类型。


x = 2.0
a = int(x)
b = complex(x)
print(a) # out: 2
print(b) # out: 2+0j

随机数 Random Number

Python 没有 random() 函数来创建随机数，但 Python 有一个名为 random 的内置模块，可用于生成随机数：


import random # 导入模块random
print(random.randrange(1,10)) # 打印一到九之间的随机数
# out: 8
