输入与输出

任何计算机程序都是为了执行一个特定的任务，有了输入，用户才能告诉计算机程序所需的信息，有了输出，程序运行后才能告诉用户任务的结果。 根据输入的不同，输出的结果也会不同。

输入(input)与输出(output)被统称为input/output(IO)。 print()与 input() 是在命令行下进行最基本的输入和输出。

输出

print('hello, world!')
print('asas','sadfd','dfas')
# output: asas sadfd dfas
print(300)
print(100 + 200) 
print("100 + 200 =", 100 + 200)
# output: 100 + 200 = 300
  • print() 表示在屏幕上输出指定的文字，但它并不会自动换行
  • 'xx'与 ''xx'' 都可以用来表示字符串
  • , 可以将多个字符串分隔开，而它的输出结果是一个空格
  • print() 也可以直接打印整数或者进行有效的数学计算

输入

  • 变量 变量之中储存了一定的内容，在计算机程序中，变量不仅可以为整数或浮点数，还可以是字符串，因此，name作为一个变量就是一个字符串。 print() 输出结果，相应的，input() 可以让用户从电脑中输入一些字符 
name = input() # 将input()存放到一个变量中，且打下这行代码且按下回车后，对本文件进行运行，便可在终端中进行输入了
print(name) # 调用name这个变量，进行输出
# input: sean
# output: sean
